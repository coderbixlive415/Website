HOW TO USE IN CANVA
1. Open Canva → New design → Custom size → 1280 x 720px
2. Divide canvas into 3 equal sections using rectangle shapes (426px each)
3. Use the background colors shown in each version above
4. Drop your real photos into each panel and use Background Remover to cut out the subject
5. Add logos as PNG transparent files on top
6. Add a thin 4px black line between each panel
7. Make sure each child's face is centered and fills most of the panel height
8. Logo should sit at the top center of each panel — big enough to read at mobile size
